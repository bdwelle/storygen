# Storyline Proposal Task

Based on the context above, propose 3-5 alternative storylines for {characters}.

{multi_character_note}

Each storyline should be a logical narrative thread that could span a chapter.

## For Each Storyline Provide:

1. **Number**: Sequential number (1, 2, 3...)
2. **Title**: Short, evocative name
3. **Thruline**: One-sentence summary of the narrative thread
4. **Arc**: Beginning → key beats → ending
5. **Length**: Approximage length of eventual story (in words)
6. **Opens with**: The scene that starts this storyline
7. **Closes with**: The scene that ends this storyline
8. **Value progression**: How the story value changes (with numbers like -4 to +3)
9. **Pro/Con**: Brief assessment of this storyline

## Storyline Requirements:

- Explore the character(s)' want vs. need tension
- Create opportunities for character arc(s)
{relationship_note}- Involve other characters meaningfully
- Fit the genre and world rules
- Could be broken into 3-5 scenes
- Each should offer a different approach or focus
