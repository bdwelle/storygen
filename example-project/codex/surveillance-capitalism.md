# Surveillance Capitalism

An economic system where human experience is commodified as raw material for behavioral data extraction, prediction, and modification.

In The Ship's context, this manifests as:
- Continuous biometric monitoring
- Behavioral prediction and manipulation
- Experience extraction as economic value
- Privacy as abolished concept
